# Tower icon assets

Put custom tower icons in this folder using these filenames:

- `firewall.png`
- `patch.png`
- `cryo.png`
- `cache.png`

Recommended size: 128x128 or 256x256 PNG with the important artwork centered. The game displays these as square icons in the tower picker. If a file is missing, the game falls back to the original F/P/C/$ text icon.
